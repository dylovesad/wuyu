# 任务二：简单 PCB 绘制

## 📃 任务描述

动手画一块降压板！

在机器人上有很多设备，他们需要的供电电压五花八门，19V 给车载 PC 供电、24V 给开发板供电、5V 给红点激光器供电等等；但电源只由一块 24V 的电池来提供。

这时候就需要结合使用 DCDC 电源 和 LDO 电源来把 24V 电压变成我们想要的比较低的各种电压。

## 🎯 任务目标

绘制一块从 24V 到 5V 的 DCDC 降压模块，并且打样把板子生产出来。
